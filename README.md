# Snake con React

Juego clásico de Snake hecho en React usando CDN y Babel, todo dentro de un solo archivo `index.html`.

## Archivos

- `index.html`: contiene toda la aplicación, estilos y lógica del juego.
- `README.md`: descripción breve e instrucciones.

## Cómo jugar

1. Abrí `index.html` en el navegador.
2. Presioná **Iniciar juego** o una tecla de movimiento.
3. Mové la serpiente con las flechas o con **WASD**.
4. Comé la comida para crecer y sumar puntos.
5. El juego termina si chocás con una pared o con tu propia serpiente.
6. Usá **Jugar otra vez** para reiniciar.

## Extras incluidos

- Pantalla de inicio
- Reinicio rápido
- Animaciones suaves
- Aumento de dificultad por niveles
